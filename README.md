# Data-Driven-Capital-Allocation-Strategy
This project delivers a multi-phase corporate decision-support framework engineered in Microsoft Excel to optimize capital allocation for a large energy utility corporation.
The core objective was to eliminate departmental biases and transition capital budgeting from an intuitive "best-guess" process to data-driven, prescriptive math. The project evolved through three distinct strategic phases: 
<li> <u> Phase 1 (Spreadsheet Architecture & Heuristic Testing)</u>: Designed a dynamic decision-support spreadsheet that normalized raw financial metrics by risk-adjusted joint-venture partnership percentages and scaled optimistic forecasts using a global downside discount. This stage established interactive conditional formatting logic gates to flag multi-year budget breaches and demonstrated why manual selection methods fail under complex constraint layers.</li>
<li> <u> Phase 2 (Simplex LP Optimization)</u>: Deployed automated mathematical modeling via Excel Solver utilizing the Simplex LP (Linear Programming) method. This phase successfully maximized total portfolio Net Present Value (NPV) to $2,800 Million, while maintaining absolute cross-functional representation and staying well under strict annual and cumulative capital spending ceilings.</li>
<li> <u> Phase 3 (Enterprise Resource Constraints)</u>: Expanded the optimization matrix to account for critical human capital constraints, incorporating a finite pool of 8 available Senior Project Managers restricted by unique, documented technical qualification boundaries.</li>
By balancing strict multi-year cash flow limitations directly alongside operational resource allocations and dynamic "What-If" sensitivity tables, this end-to-end framework bridges the gap between data analytics and high-level strategic enterprise governance.
